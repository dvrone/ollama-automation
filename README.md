## 📖 About

**Ollama Automation** is a documentation-based repository that teaches how to build AI-powered automation using **Python** and **Ollama**.

The repository focuses on practical learning through tutorials and hands-on projects instead of providing a ready-made application.

---

## ✨ What You'll Learn

* Installing and configuring Ollama
* Using the Ollama Python library
* Creating a terminal AI chatbot
* Working with streaming responses
* Managing conversation history
* Prompt engineering
* AI automation with Python
* Building local AI assistants
* Integrating LLMs into Python projects

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone git@github.com:dvrone/ollama-automation.git
cd ollama-automation
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Windows:

```powershell
.venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Install Ollama

Download and install Ollama from the official website, then download your preferred model, for example:

```bash
ollama run qwen2.5:0.5b
```

### 5. Start Learning

Open the documentation:

* 📖 **Tutorial:** `docs/tutorial.md`
* 🚀 **Projects:** `docs/projects.md`
